# Memo App

シンプルで安全なメモ共有サービス

## 機能

- メモの作成と共有
- パスワード保護（オプション）
- カスタムURL（オプション）
- エンドツーエンド暗号化
- ログ・個人情報の記録なし

## セットアップ

```bash
npm install
npm start
```

サーバーはポート3000で起動します。

## デプロイ

Coolifyでのデプロイに対応しています。

環境変数:
- `PORT`: サーバーポート（デフォルト: 3000）

## 技術スタック

- Node.js + Express
- SQLite
- AES-256-CBC暗号化
